## Breakpoints

| Breakpoint | Target | Changes |
|---|---|---|
| `max-width: 1024px` | Tablet and below | Right panel width 360px, grid columns smaller, product card padding reduced |
| `max-width: 767px` | Mobile | Layout becomes column, left/right max-height 50vh, all sizes reduced |
| `max-width: 480px` | Small phones | Left max-height 45vh, right 55vh, grid 2 columns, further size reductions |
